This repository contains part of the group project that is still under development (unfinished).
The project consists in a website designed to teach people Machine Learning and allows the user to 
interactively tweak the parameters of ML models and compare the behaviour of different models with 
different parameters. 

It has been done collaboratively in a team of 6 people using GitLab. 
The framework is Django.
This folder is just a part of the whole project.
